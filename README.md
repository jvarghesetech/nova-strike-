# 🚀 Nova Strike

A fast-paced neon space shooter built with vanilla HTML5 Canvas and JavaScript — no frameworks, no dependencies, no external assets. Everything from the graphics to the sound effects is procedurally generated in the browser.

## [🕹️ Play Live Demo](https://jvarghesetech.github.io/nova-strike/)

## 🎮 Gameplay

Pilot your ship through endless waves of increasingly dangerous enemies. Chain kills to build combos, grab powerups to stay alive, and see how long you can survive.

### Controls

| Input | Action |
|---|---|
| `WASD` / `Arrow Keys` | Move |
| `Mouse` | Aim |
| `Left Click` | Fire |
| `Shift` + Direction | Dash (i-frames) |

### Enemy Types

- **🔴 Basic** — Standard pursuit enemies
- **🟡 Fast** — Quick and agile, low HP
- **🟣 Tank** — Slow but heavily armored with visible HP rings
- **🔵 Shooter** — Fires projectiles back at you

### Powerups

- **💚 Heal** — Restores 30 HP
- **💗 Spread** — 5-way spread shot for 8 seconds
- **💛 Rapid** — Doubled fire rate for 8 seconds

## ⚡ Features

- Zero dependencies — pure HTML5 Canvas + vanilla JS
- Procedural audio (Web Audio API) — all SFX generated in real-time
- Combo system with score multipliers
- Dash mechanic with invulnerability frames
- Screen shake, particle systems, and thruster trails
- Progressive wave difficulty with dynamic enemy spawning
- Custom crosshair and parallax starfield
